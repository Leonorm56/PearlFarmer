## Quick Start

### Prerequisites
- Node.js (v18 or higher)
- pnpm package manager
- Chrome browser

### Installation
git clone https://github.com/Leonorm56/NileChain.git
cd nilechain
pnpm install

### Build
pnpm build:farmer

### Release
node release.js
